# Restaurant Management System

## Description
This project is a console-based restaurant management system developed for the Software Engineering course.

## Functions
- Customer: view menu, create order
- Staff: manage orders, process payment
- Admin: manage menu, view reports

## Technologies
- Python
- SQLite
- Docker
- GitHub
